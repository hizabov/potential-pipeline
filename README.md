# potential-pipeline
Creating my own animation and vfx pipeline

The plan is to create an animation and vfx pipeline with the following directory structure.

root
- project_name
  - assets
    - characters
      - character_name
        - modeling
        - rigging
        - texturing
    - props
      - prop_name
        - modeling
        - rigging
        - texturing
    - sets
      - set_name
        - modeling
        - rigging
        - texturing
  - shots
    - sequence_number
      - shot_number
        - animation
        - lighting
        - compositing
